# üß† Sentiment Analysis (Python + Flask)

A **machine learning-based Sentiment Analysis** project that analyzes live or static text data to determine whether the sentiment is **Positive**, **Negative**, or **Neutral**.  
This project integrates **Python**, **Tweepy**, **VADER**, and **Flask** to fetch tweets, perform sentiment analysis, and visualize the results on a simple web interface.

---

## üöÄ Features
- Real-time tweet extraction using **Tweepy (Twitter API)**  
- Sentiment classification using **VADER Sentiment Analyzer**  
- Web dashboard built with **Flask**  
- Visualization of sentiment distribution (Pie chart / Bar chart)  
- User input for keyword-based sentiment search  

---

## üß© Tech Stack
- **Python 3.x**  
- **Flask** (for web interface)  
- **Tweepy** (for fetching tweets)  
- **VADER / NLTK** (for sentiment analysis)  
- **Matplotlib / Plotly** (for visualizations)  
- **HTML, CSS, JS** (frontend UI)
